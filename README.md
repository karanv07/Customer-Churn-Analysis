# Customer-Churn-Analysis

## Project Overview
This project analyzes telecom customer churn using Python. The goal is to identify factors influencing customer attrition and generate insights that help businesses improve customer retention.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Around 26.5% of customers churned.
- Month-to-month contracts show the highest churn (~45%).
- Fiber optic users churn more (~41%) than DSL users (~19%).
- Senior citizens show higher churn (~41%).
- Electronic check payment method has the highest churn (~45%).

## Business Recommendations
- Encourage long-term contracts.
- Improve early customer onboarding experience.
- Promote automatic payment methods.
- Offer value-added services like tech support and security.
